## Why It Matters
系统性地将Query Engine工程师的背景迁移到AI时代，不是从零开始，而是能力迁移。

## Key Insights

### 1. 能力迁移而非身份重置
Query Engine工程师的系统能力（执行调度、资源管理、缓存优化）与AI系统（推理服务、RAG、Agent）高度相通。

### 2. 四个核心能力层次
1. **会用** - 将AI作为生产力工具
2. **会懂** - 理解LLM、RAG、Agent的基本原理
3. **会做** - 设计并实现完整AI系统
4. **会想** - 理解AI对职业路径的影响

### 3. 最值得关注的方向
- LLM推理系统与AI基础设施（Continuous Batching, Paged Attention, KV Cache）
- RAG与检索增强系统（与数据库天然契合）
- AI for Engineering（提升研发工作流）
- Agent与工作流自动化
