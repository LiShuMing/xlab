## 重要更新

### [ReadySet] GitHub - readysettech/readyset: Readyset is a MySQL and Postgres wire-compatible caching layer

**变更内容：**
- Introduced incremental updates for cached select statements
- Provides horizontal read scaling without application code changes

**重要性：**
- Reduces latency for repeated queries without full materialized view maintenance overhead
- Enables legacy database scaling without migration to new OLAP systems

**重要性：**
